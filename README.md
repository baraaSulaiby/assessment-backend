# Products CLI CRUD

A simple CLI application built with Python, SQLAlchemy, Click, and PostgreSQL.

## Features
- List all products
- Add a product
- Get product by ID
- Update a product
- Delete a product

## Tech Stack
- Python
- SQLAlchemy
- PostgreSQL
- psycopg2-binary
- python-dotenv
- Click
